# Azure Fundamentals

## What is the AZ-104?
- The Azure Administrator Certification is for system admins or those looking to become system admins with Azure/Microsoft workloads.
- The Azure Administrator will demonstrate a person can manage and configure devices, users, permissions, general computing, cloud storage and cloud networking.
- The course code is AZ-104 and is the natural path after Azure Fundamentals AZ-900 and for those looking to obtain the Azure Solutions Architect AZ-303
- Ideal for those seeking a cloud role that is more configuration based than having to write computer code.

## AZ vs AWS cs GCP
- 2x more difficult than AWS Solution Architect Associate
- 3x more difficult than GCP cloud engineer
- Azure exams are generally more difficult beyond the fundamenal certification because they will test for `practical knowledge` with exact steps on how to use a service instead of conceptual or strategic information.
- An example would be, how do you setup an Azure File Shared sync? This kind of question would be with mutiple choice in the right order.

## Content Outline
- `15-25%` Manage Azure Identities and Governance (IAM)
- `15-20%` Implement and manage `storage`
- `20-25%` Deploy and manage `Azure compute resources`
- `25-30%` Configure and manage `virtual networking`
- `10-15%` Monitor and backup Azure resources

> Passing Grade is *700/1000.
- You need to get "around" 70% to pass

### Response Types
- There are 40-60 questions.
  - Some questions are worth more than 1 point
  - There is no penalty for wrong questions
  - Some questions `can NOT be skipped`
- Format of questions:
  - Multiple choice
  - Multiple answer
  - Drag and drop
  - Hot area
  - Case Studies

> Exam duration is 180 minutes so ~1 minute per question. Valid for 24 months.
